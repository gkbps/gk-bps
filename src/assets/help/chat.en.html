Chat

Chat is a small tool for users to interact with others in a realtime basic.

Chat usage is straightforward, it allows user to invite others to join a private room.
The room could be newly created or an existing one. The number of room members are unlimited.

Invite a contact to join an existing chat room

Create a new room to chat with the selected contact.

In order to protect privacy, chat history will not be stored and users are only
able to view exchanged information amongst chatters once users joined and are in the room.
