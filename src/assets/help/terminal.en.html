Terminal

A text based user interface that allows user to access into hidden information.

Main purpose of terminal is for error auditing or bug tracking at user side.

In addition, thanks to Terminal that could at the same time serve best user experience and supports

Below lists typical commands:

  1. date To show today information
  2. show
    • [error] To show list of error(s) during system operation
      • [-] To show list of all errors
      • [last] To show the last error
    • [lang] To show the current language
    • [wklge] To show working legal entity
    • [wkyear] To show working year
