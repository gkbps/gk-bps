View Changes

Guidelines to read changes of document

  • Kind - indicates the kind of change; will be one of the following:
    • N - indicates a newly added property/element
    • D - indicates a property/element was deleted
    • E - indicates a property/element was edited
    • A - indicates a change occurred within an array
      • index - when kind === 'A', indicates the array index where the change occurred
      • item - when kind === 'A', contains a nested change record indicating the change that occurred at the array index
  • Info - the property path (from the left-hand-side root)
  • Before - the value on the left-hand-side of the comparison (undefined if kind === 'N')
  • After - the value on the right-hand-side of the comparison (undefined if kind === 'D')
Changes are arranged in the order of recent event
